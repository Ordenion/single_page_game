# SHIP IT: Code Roast

A short, energetic coding mini-game for teams, meetups, and learning groups.

This is not a strict “knowledge test”. It’s a quick interactive round where people laugh, debate answers, and recognize real-world coding gotchas. In 8 minutes, one player goes through short JS/Python questions, can use hints with a score penalty, and finishes with a playful title.

## Core Idea

- One screen, one phone/laptop, one player per round.
- Fast pace: timer, score, instant feedback.
- Friendly, ironic tone: roast vibe without toxicity.

## Why It Works

- Engages even people who usually stay quiet in calls.
- Reduces fear of “making mistakes in code”.
- Great warm-up before workshops, interview days, or internal demos.

## Where It Fits

- Inside teams (retros, Friday activities, onboarding).
- In learning groups (quick pre-class warm-up).
- At events (short interactive segment between talks).

## Proposal Angle

Position it as “edutainment for developers”:
a fast game block that entertains while reinforcing practical engineering thinking.
